# LazyVim configuration

These are just my personal vim configuration files.
Feel free to use them but please don't blame me if something goes wrong.

## Requirements

- neovim (0.10.0+)
- ripgrep
- nodejs
- fzf (optional)
- fd (fd-find)
- fish (fish indent)
- cmake
- npm
- cargo

### Java (optional)

- maven
